# Draw2.1

## Project Overview
Draw2.1 is an innovative graphics rendering application designed to make drawing and design easy and accessible for everyone. With intuitive features and a user-friendly interface, users can unleash their creativity with a variety of tools and functionalities.

## Quick Start Guide
To get started with Draw2.1:
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/soulde10-prog/Draw2.1.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Draw2.1
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Features
- **Intuitive User Interface**: Designed for ease-of-use.
- **Multiple Drawing Tools**: Includes brushes, shapes, and text tools.
- **Undo/Redo Functionality**: Easily correct mistakes.
- **Layer Support**: Manage multiple layers in your designs.

## Prerequisites
Before you begin, ensure you have met the following requirements:
- You have a compatible system (Windows, macOS, or Linux).
- Node.js is installed on your machine. You can download it from [Node.js Official Website](https://nodejs.org/).

## Installation
Follow these steps to install Draw2.1:
1. Once you have cloned the repo and navigated to the directory, run the following command:
   ```bash
   npm install
   ```
2. After the installation is complete, you can start the application using:
   ```bash
   npm start
   ```
