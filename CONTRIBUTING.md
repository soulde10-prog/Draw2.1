# Contribution Guidelines

Welcome to the Draw2.1 project! We appreciate your interest in contributing. Please follow the guidelines below to help us maintain a high-quality project.

## Reporting Bugs
- If you encounter a bug, please check the existing issues to see if it has already been reported.
- Create a new issue if it hasn't been reported, including steps to reproduce, the expected outcome, and any relevant screenshots or logs.

## Suggesting Enhancements
- If you have an idea for enhancing the project, please create a new issue with a detailed description of the enhancement.
- Include the reasoning behind your suggestion and any examples where applicable.

## Development Setup Instructions
1. **Clone the repository**:
    ```bash
    git clone https://github.com/soulde10-prog/Draw2.1.git
    cd Draw2.1
    ```
2. **Install dependencies**:
    ```bash
    # Use the appropriate command for your setup, e.g.
    npm install  # For Node.js projects
    ```
3. **Run the project**:
    ```bash
    # Use the appropriate command to start the project
    npm start
    ```

## Making Changes
- Create a new branch for your feature or bugfix:
    ```bash
    git checkout -b my-feature-branch
    ```
- Make your changes and commit them:
    ```bash
    git commit -m "Add new feature or fix bug"
    ```

## Submitting Pull Requests
1. **Push your branch** to the repository:
    ```bash
    git push origin my-feature-branch
    ```
2. **Create a pull request**:
    - Go to the repository on GitHub and you will see an option to create a pull request for your branch.
    - Provide a clear description of what your pull request does.

Thank you for contributing to Draw2.1!
